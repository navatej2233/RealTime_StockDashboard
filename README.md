## Description
The **Real-Time Stock Market Dashboard** is a Python-based web application that fetches and visualizes live stock data.  
It displays stock prices, trends, and key financial indicators in real-time using Streamlit for the user interface and APIs for fetching live data.

This project allows users to monitor multiple stocks simultaneously with dynamic charts and analytics.

---

## Features
-  Real-time stock price tracking  
-  Interactive charts and visualizations  
-  Search and view multiple stock symbols  
-  Displays historical trends and daily updates  
-  Built with Streamlit for quick and responsive UI  

---

##  Technologies Used
- **Python 3.14**
- **Streamlit** – For interactive web dashboard  
- **Requests / yfinance / APIs** – For fetching live market data  
- **Pandas** – Data analysis and transformation  
- **Plotly / Matplotlib** – Graph plotting and visualization
